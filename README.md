## Getting Started


Follow the instructions to run the project on your device locally.

### Prerequisites


* Django
  ```sh
  pip install django
  ```
* Pillow
  ```sh
  pip install pillow
  ```
 

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Samyak-jain09/CourseRoom.git
   ```
2. Enter the virtual environment.
   ```sh
   pipenv shell
   ```
4. Launch the project
   ```sh
   python manage.py runserver
   ```
